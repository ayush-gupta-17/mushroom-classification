# Mushroom Classification using Machine Learning

## Project Overview

This project is a basic Machine Learning application that classifies mushrooms as **Edible** or **Poisonous** based on their physical characteristics. The model is trained using the Mushroom Dataset and uses a **Random Forest Classifier** for prediction.

## Features

* Data preprocessing using Label Encoding
* Train-test split for model evaluation
* Random Forest Classification
* Accuracy evaluation
* Console-based user input for prediction
* Single Python file implementation

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Dataset

* Mushroom Dataset
* Target Classes:

  * **e** – Edible
  * **p** – Poisonous

## Installation

Install the required libraries:

```bash
pip install pandas scikit-learn
```

## How to Run

```bash
python mushroom.py
```

The program will:

1. Load the dataset.
2. Train the Random Forest model.
3. Display the model accuracy.
4. Ask the user to enter mushroom characteristics.
5. Predict whether the mushroom is **Edible** or **Poisonous**.
